# Gold Rush Lines — Game Rules v0.1

## Game Overview

Gold Rush Lines is a classic 5-reel, 3-row video slot with 20 fixed paylines.

The game is currently in v0.1 development state and includes only the base game. Free Spins and Bonus Buy are disabled for this version.

## Game Type

- Type: Video Slot
- Win Type: Lines
- Reels: 5
- Rows: 3
- Paylines: 20
- Direction: Left to right
- Current RTP target: approximately 94–95%
- Max Win: 1000x
- Current version: v0.1 Base Game

## Symbols

### Special Symbols

- W = Wild Gold
- S = Scatter Bonus

### High Symbols

- H1 = Chest
- H2 = Crown
- H3 = Gem
- H4 = Ring

### Low Symbols

- L1 = A
- L2 = K
- L3 = Q
- L4 = J
- L5 = 10

## Wild Rules

Wild substitutes for all regular symbols.

Wild does not substitute Scatter.

In v0.1, Wild only pays on 5 of a kind.

## Scatter Rules

Scatter exists in the reel strips, but Free Spins are disabled in v0.1.

Free Spins will be added later in v0.2.
